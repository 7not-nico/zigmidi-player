# SoundFont Files

Place your SoundFont files (.sf2) in this directory.

The player uses `standard-midisf.sf2` as the default SoundFont if no SoundFont is specified.

You can download free SoundFonts from:
- https://musical-artifacts.com/artifacts/538 (FluidR3_GM.sf2)
- https://www.zrythm.org/downloads.html

Example:
- default.sf2
- piano.sf2
- orchestra.sf2
